# Date Chart

Creates a SVG date chart from CSV table like this:

| Date                         | Series 1 | Series 2 | ... |
| ---------------------------- | -------- | -------- | --- |
| Tue Apr 28 14:38:44 UTC 2020 | 76.67    | 66.42    | ... |
| ...                          | ...      | ...      | ... |

Written using **purity**.

# Getting started

Run compiler: `bash compile.sh`. Run server on port `:8090`: `bash serve.sh`.
